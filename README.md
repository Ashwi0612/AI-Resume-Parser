# AI-Based Resume Parser and Candidate Ranking System

## Project Overview

This project uses Artificial Intelligence (AI), Natural Language Processing (NLP), and Machine Learning techniques to automatically analyze resumes and classify them into suitable job categories.

Recruiters often receive hundreds of resumes for a single job opening. Manually reviewing all applications is time-consuming. This system helps automate the screening process by analyzing resume content and predicting the most relevant job domain.

---

## Objectives

* Automate resume screening.
* Classify resumes into job categories.
* Extract meaningful information from resume text.
* Rank candidates based on relevant skills.
* Reduce recruiter workload.

1. Data Collection
2. Data Preprocessing

6. Train-Test Split
7. Model Training
8. Model Evaluation
9. Resume Classification
10. Candidate Scoring

---

## Algorithm Used

### TF-IDF Vectorization

TF-IDF converts textual resume data into numerical feature vectors suitable for machine learning.

### Logistic Regression

Logistic Regression is used for resume classification because it is efficient, fast, and performs well on text classification tasks.
